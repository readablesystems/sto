Detailed Description

template<unsigned int Timeout, class Duration = std::chrono::milliseconds>
class cds::backoff::delay_of< Timeout, Duration >

Delay back-off strategy, template version.

This is a template version of backoff::delay class. Template parameter Timeout sets a delay timeout. The declaration cds::backoff::delay_of< 5 > bkoff is equal for cds::backoff::delay<> bkoff(5).
